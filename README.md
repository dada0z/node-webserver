# node-webserver
A simple web server by nodejs

usage
---------------------------
[XXX@XXX node-webserver]$node server.js

config.json
---------------------------
```
{
  "port":"",
  "rootdir":""
}
```
port will be 8080 if not given.

rootdir can be relative path or absolute path.
